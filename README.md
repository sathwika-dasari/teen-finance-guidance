# Teen Finance Guidance

A premium full-stack web application designed to empower individuals with financial literacy and income guidance.

## Features
- **Age-Based Routing**: Custom learning paths for users under and over 18.
- **Teens (< 18)**: Basics of finance, scam prevention, and pocket money budgeting.
- **Adults (> 18)**: Skills-based part-time guidance, stocks, and advanced budgeting.
- **Progress Tracking**: Monitor your learning journey via a dynamic dashboard.
- **Premium UI**: Dark mode with glassmorphism and smooth animations.

## Tech Stack
- **Backend**: Flask (Python), SQLite
- **Frontend**: HTML5, Vanilla CSS, Javascript
- **Architecture**: Modular structure with a Rule Engine for personalization.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Initialize the database: `python init_db.py`
3. Run the application: `python backend/app.py`
4. Open `http://localhost:5000` in your browser.
